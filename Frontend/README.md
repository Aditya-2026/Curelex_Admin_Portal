# Curelex Admin Portal (Frontend)

This project is the **Admin Dashboard UI** for the Curelex platform.

The dashboard allows administrators to monitor platform analytics, manage doctors and patients, and review doctor verification requests.

---

# Technologies Used

HTML
CSS
JavaScript
Chart.js

---

# Features Implemented

Admin Dashboard with statistics cards
Patient analytics charts
Doctor verification panel
Doctors management page
Patients management page
Search and filter functionality
Responsive admin layout

---

# Project Structure

assets/ → Images and logo files

css/ → Main stylesheet

js/ → JavaScript logic for dashboard and modules

HTML Pages:

dashboard.html – Main dashboard

analytics.html – Patient analytics charts

doctors.html – Doctors management

doctor-verification.html – Doctor verification panel

patients.html – Patients management

settings.html – Admin settings

---

# API Integration Notes

The frontend currently uses **static sample data** for UI demonstration.

Backend developers should replace the static arrays with API calls.

Suggested APIs:

GET /api/dashboard-stats

GET /api/doctors

GET /api/patients

GET /api/doctor-applications


---

# Charts

Charts are implemented using **Chart.js**.

Chart data is currently static and should be replaced with backend analytics data.

Files containing chart logic:

js/dashboard.js
